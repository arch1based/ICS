# Οδηγός Εγκατάστασης & Λειτουργίας — Sunmi CPad + NT320

Αναλυτικός οδηγός για την εγκατάσταση και λειτουργία του bundle **Sunmi CPad 11"** με τη **Βάση Εκτύπωσης C14004118** και τον θερμικό εκτυπωτή **NT320 80mm**.

## Περιεχόμενα οδηγού

- **Σύνδεση CPad με τη Βάση** — Pogo PIN (6+8), quick-lock, χωρίς καλώδια
- **Ασύρματη Σύνδεση Εκτυπωτή** — WiFi (LAN/IP) ή Bluetooth 5.3 BLE (~10m)
- **Λειτουργία Reverse Charging** — Αδιάλειπτη λειτουργία εκτυπωτή από μπαταρία CPad (8.000 mAh) σε διακοπή ρεύματος

## Τεχνικά Χαρακτηριστικά

| | |
|---|---|
| Tablet | Sunmi CPad 11" · Android 14 |
| Βάση | Printing Base C14004118 · Pogo PIN (6+8) |
| Εκτυπωτής | Sunmi NT320 80mm · 150 mm/s · 203×203 dpi |
| Συνδεσιμότητα | USB-C · LAN · WiFi 802.11ax · Bluetooth 5.3 BLE |
| Reverse Charging | Μέσω Pogo PIN · Κατώτατο όριο μπαταρίας 20% |
