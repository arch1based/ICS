# Οδηγός Ρύθμισης Εκτυπωτών Xprinter μέσω Android (Xtest)

Αναλυτικός οδηγός βήμα προς βήμα για τη σύνδεση και ρύθμιση θερμικών εκτυπωτών **Xprinter** μέσω της εφαρμογής **Xtest** σε συσκευές Android.

## Περιεχόμενα οδηγού

- Εγκατάσταση της εφαρμογής Xtest από το Google Play Store
- Σύνδεση εκτυπωτή μέσω δικτύου (LAN/IP) με αυτόματη ή χειροκίνητη ανάθεση IP
- Επισκόπηση των βασικών λειτουργιών της εφαρμογής
- Ρύθμιση ελληνικού code page (PC737) για σωστή εκτύπωση ελληνικών χαρακτήρων
